Crie requisições HTTP e teste sua API RESTful de forma simples

Resquest Builder é uma ferramenta para simulação de requisições HTTP. Você pode:

  • Simular requisições GET, POST, PUT, DELETE, PATCH e HEAD.
  • Enviar dados por REQUEST PAYLOAD ou FORM ENCODE
  • Enviar headers personalizados
  • Enviar dados personalizados
  • Salvar requisições para enviar posteriormente

Requisitos

É necessário ter Node.js e NPM instalados

Como instalar

git clone https://github.com/douglaszporto/request-builder.git
cd request-builder
>npm install
npm start
